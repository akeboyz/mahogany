# Web App UI/UX Design - Digital Signage Operations System

## Overview
Internal web application for managing digital signage operations from opportunity to deployment and ongoing maintenance.

**Tech Stack Recommendation:**
- Frontend: Next.js 14 (App Router), React, TypeScript
- Styling: TailwindCSS
- Backend: Firebase (Firestore, Storage, Functions)
- Forms: React Hook Form + Zod validation
- Tables: TanStack Table
- Charts: Recharts or Chart.js
- PDF Generation: jsPDF or react-pdf
- Authentication: Firebase Auth

### Key Features to Implement

1. **Authentication & Authorization**
   - Firebase Auth
   - Role-based access (Sales, Operations, Finance, Manager, Admin)
   - Row-level permissions

2. **Real-time Updates**
   - Firestore real-time listeners
   - Auto-refresh data on updates
   - Optimistic UI updates

3. **Form Management**
   - React Hook Form for performance
   - Zod schema validation
   - Auto-save drafts (localStorage + Firestore)
   - Multi-step forms with progress

4. **Data Tables**
   - TanStack Table for sorting, filtering, pagination
   - Column visibility toggle
   - Export to CSV/Excel
   - Bulk actions

5. **PDF Generation**
   - Generate quotations
   - Generate contracts
   - Generate reports
   - Email integration

6. **File Upload**
   - Firebase Storage
   - Drag & drop
   - Image preview
   - File size validation

7. **Notifications**
   - Toast notifications (react-hot-toast)
   - Email notifications (Firebase Functions)
   - Push notifications (optional)

8. **Search**
   - Global search across all entities
   - Algolia integration (optional for advanced search)
   - Recent searches

9. **Audit Trail**
   - Log all changes
   - Show change history
   - User attribution

10. **Performance**
    - Server components where possible
    - Image optimization
    - Code splitting
    - Lazy loading
